矿用跨轨式智能巡检机器人

本项目是一款应用于煤矿井下带式输送机巡检的智能机器人，结合嵌入式控制、图像识别与多传感器监测，实现了设备故障预警与环境感知。
🌟 功能简介
LED 闪烁与流水灯控制
按键输入控制风扇启停
串口通信与上位机调试
 动能回收与多阶段充电（扩展设计）

 🧱 系统架构
硬件：开发板、电机、风扇、传感器模块
软件：C语言嵌入式开发（含GPIO、PWM、串口、状态机）
可拓展：YOLO视觉识别、电池管理、云端联动

📁 项目结构
 `/hardware`：原理图、电路图、实物照片
 `/software`：底层驱动与主控逻辑
 `/docs`：设计说明书、答辩材料
 `/images`：流程图与展示图
